Efficient CNN with Depthwise Separable Convolutions for Image Classification

Deep Learning TensorFlow CV

Developed a CNN architecture for efficient image classification using depthwise separable convolution, achieving 93-97% accuracy on CIFAR10 while maintaining low computational complexity. The model includes features like two-stage 1x1 convolution for channel reduction, patch-based image representation. Created three variants (Small: 130K params/0.03 GFLOPS, Medium: 490K params/0.12 GFLOPS, Large: 999K params/1.01 GFLOPS) demonstrating different speed-accuracy tradeoffs.
